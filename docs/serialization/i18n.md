# Локализация

Формат локализации – [i18n](http://i18njs.com).

## Схема локализованных данных

Локализованный объект в формате i18n или

```json
{"ref": # сссылка на локализованный объект}
```

В случае, если объект не найден по ссылке на i18n-файл или схема не соответствует ожидаемой при локализации должен быть возвращена ссылка или объект соответственно
